1. Objet du service

Wagon est une application permettant d’obtenir des informations sur les transports, les arrêts à proximité et les trajets en cours, ainsi que de signaler des informations utiles aux autres utilisateurs.

2. Accès au service

3. Responsabilités

L’utilisateur est responsable des informations qu’il partage via les signalements. L’éditeur ne saurait être tenu responsable des erreurs, omissions ou indisponibilités du service.

4. Utilisation des données

Politique de confidentialité

Wagon est conçue pour utiliser le moins de données personnelles possibles. Nous envoyons le minimum de données possibles afin de fournir un service rapide et fiable.

Les données que nous collectons

Votre adresse IP est collectée à chaque requête afin d’éviter les abus sur le service. Les données sont conservées pendant quelques heures uniquement.

Les données que nous utilisons

Dans votre utilisation de l’app, Wagon récupère votre localisation pour afficher les arrêts à proximité ou le trajet en cours. Ces données ne sont pas stockées.

Vos signalements

Pour assurer des signalements authentiques, nous collectons les informations suivantes pendant une durée maximale d’un an :

  • Adresse IP (sous forme d’identifiant anonymisé)
  • Identifiant unique généré aléatoirement
  • Position approximative du signalement
  • Arrêt concerné
  • Trajet
  • Ligne empruntée

Ces données ne permettent pas de vous identifier directement.

Utilisation depuis ChatGPT (OpenAI)

L’application est disponible sur le magasin d'applications intégré à ChatGPT.

Aucune donnée personnelle supplémentaire n’est collectée par Wagon dans ce cadre, en dehors des informations que l’utilisateur saisit volontairement (par exemple : point de départ et d’arrivée d’un itinéraire).

Certaines données techniques envoyées par OpenAI peuvent être utilisées :

  • Les informations saisies par l’utilisateur
  • Une position approximative (si disponible), pour les analyses et pour fournir un point de départ d'itinéraire
  • Des identifiants de session techniques

Ces données peuvent être utilisées pour améliorer le service, effectuer des analyses ou proposer des suggestions pertinentes.
